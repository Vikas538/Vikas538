## 🔥 Featured Work

### OrbIT — Autonomous Jira-to-Code Agent Platform
> Built at Electronic Arts | Got director approval for sprint inclusion

An autonomous multi-agent platform that takes a Jira ticket and writes, tests, and commits production code — without human intervention unless needed.

**Architecture:**
- ECS Fargate + SQS for scaling to 100+ concurrent agents
- Containerized execution with isolated sandboxes per task
- Human-in-the-loop approval gate for package installs and risky changes
- ChromaDB vector memory for cross-ticket context
- WebSocket terminal UI with real-time git diff viewer
- Reviewer + orchestrator agent pattern for multi-file conflict resolution

**Stack:** Python · FastAPI · Docker · AWS ECS · SQS · ChromaDB · WebSockets
